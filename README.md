# 🍕 Dapur Putri - Web Ordering System

Proyek ini adalah sistem pemesanan berbasis web yang dibangun untuk digitalisasi UMKM **Dapur Putri**. Fokus utama proyek ini adalah menggabungkan estetika desain (DKV) dengan fungsionalitas teknologi.

### ✨ Fitur Utama
- **UI/UX Design:** Desain antarmuka yang bersih dan responsif untuk pengguna mobile.
- **Order System:** Integrasi sistem pemesanan yang langsung terhubung untuk efisiensi operasional.
- **Live Deployment:** Dihosting menggunakan Netlify untuk akses cepat dan stabil.

### 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript.
- **Deployment:** Netlify.
- **Design Tools:** Adobe Photoshop & Illustrator untuk aset visual.
